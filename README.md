# Task 11.25.2025

## Description

This task involves creating a responsive HTML and CSS student table.

The requirements are:

- Create an HTML table listing students with columns:
  - Number
  - Full Name
  - Course
  - Student ID
  - Year

- Use CSS to style the table with borders, different text styles, and tooltips.

- Implement Media Queries to make the table responsive:
  - On desktop (web view), show the full table with all columns.
  - On mobile view (screen width less than 600px), display only two columns (Number and Full Name) and hide the others for better readability.

- Apply CSS3 features and animations to enhance the design and usability.

## Summary

This project helps practice:

- HTML table creation
- CSS styling including borders, fonts, and animations
- Responsive web design using Media Queries
- Accessibility improvements like tooltips
